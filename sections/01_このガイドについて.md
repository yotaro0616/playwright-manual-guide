## Claude Code × Playwright でできること

このガイドでは、**Claude Code**（AI アシスタント）と **Playwright**（ブラウザ自動操作ツール）を組み合わせて、Web アプリのドキュメント作成を自動化する方法を学びます。

具体的には、以下の 3 つの作業を AI に任せられるようになります。

| 作業 | 使う Skill | やること |
|------|-----------|---------|
| **調査** | web-app-explorer | Web アプリの画面・機能を自動で巡回し、調査レポートを作成 |
| **執筆** | operation-manual-creator | 調査結果をもとに操作マニュアルを自動生成 |
| **撮影** | manual-screenshot-capture | マニュアル内の指示に従いスクリーンショットを自動撮影 |

通常、操作マニュアルを作るには「画面を 1 つずつ開いてメモを取り、手順を書き起こし、スクリーンショットを 1 枚ずつ撮る」という地道な作業が必要です。このガイドの方法を使えば、AI がブラウザを操作してこれらを一括で行います。

## 前提条件

### あなたに必要なもの

- **PC**（Mac / Windows / Linux いずれか）
- **インターネット接続**
- **クレジットカード**（Claude Pro プランの支払いに使用）

### 前提とする知識

- PC の基本操作（ファイル・フォルダの概念、ブラウザ操作）
- テキストエディタの基本的な使い方
- Markdown の基礎（見出し `#` や箇条書き `-` 程度）

### 前提としない知識

以下の知識は不要です。このガイドで必要な分だけ説明します。

- プログラミング経験
- ターミナル（コマンドライン）の操作
- Git / GitHub の知識

## 所要時間の目安

| セクション | 目安 |
|-----------|------|
| セクション 2〜5（準備） | 約 30〜60 分 |
| セクション 6〜8（実践） | アプリの規模による（小規模なら 1〜2 時間） |

大半の時間は AI が作業するのを待つ時間です。あなたが手を動かすのは、主にコマンドのコピー＆ペーストと結果の確認です。
